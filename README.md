# API Security Risk Analysis

## Overview

This project contains an API Security Risk Analysis completed as part of the Future Interns Cyber Security Internship.

The assessment focuses on analysing a public REST API, identifying common security risks and translating technical findings into business-friendly recommendations.

## Tools Used

- Postman
- Browser Developer Tools
- Microsoft Word
- GitHub

## Methodology

- Selected the JSONPlaceholder API
- Reviewed API documentation
- Performed read-only GET requests
- Inspected response data and HTTP headers
- Identified potential security risks
- Assessed business impact
- Provided remediation recommendations

## Key Findings

- Unauthenticated API Endpoint
- Excessive Data Exposure
- Information Disclosure Review
- Graceful Handling of Invalid Requests
- Response Header Analysis
